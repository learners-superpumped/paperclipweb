# Design Spec: paperclipweb

**Date:** 2026-03-31
**Designer:** @product-designer
**Stack:** Next.js 15 (App Router) + Tailwind CSS + shadcn/ui pattern

---

## Design System

### Colors

| Token | HEX | Usage |
|-------|-----|-------|
| Primary | #4F46E5 (Deep Indigo) | CTAs, active states, brand accent |
| Primary-50 | #EEF2FF | Icon backgrounds, hover states |
| Secondary-800 | #0F172A | Headings, primary text |
| Secondary-500 | #475569 | Body text, descriptions |
| Secondary-400 | #94A3B8 | Muted text, placeholders |
| Accent | #10B981 (Emerald) | Success states, checkmarks, running status |
| Destructive | #F43F5E (Rose) | Errors, danger zone, delete actions |
| Background | #FAFAFA (Snow) | Page background |
| Surface | #FFFFFF | Cards, modals |

### Typography

- **Font:** Inter (Google Fonts)
- **Headings:** Inter Bold/Semibold, tracking-tight
- **Body:** Inter Regular, text-sm (14px), line-height 1.5-1.75
- **Scale:** text-xs (12px), text-sm (14px), text-base (16px), text-lg (18px), text-2xl (24px), text-4xl (36px)

### Component Tokens

- **Border radius:** lg (12px), md (8px), sm (6px)
- **Border color:** secondary-200 (#E2E8F0)
- **Shadow:** shadow-sm (cards), shadow-lg (pricing popular), shadow-xl (hero mockup)
- **Transition:** 150-200ms ease for color/opacity, 500ms for progress bars

### Style

- Flat design with minimal shadows
- No gradients on backgrounds (gradient only in chart fills at 20% opacity)
- Clean hover states using color/opacity transitions (no scale transforms)
- All interactive elements have cursor-pointer
- SVG icons only (Lucide React) -- no emojis

## Pages

### 1. Landing Page (/)
- **Navbar:** Fixed, bg-white/80 backdrop-blur, max-w-6xl
- **Hero:** Centered, Badge + H1 + subtitle + 2 CTAs + "no credit card" note + dashboard mockup
- **Features:** 3-column grid with icon cards + Before/After comparison callout
- **Pricing:** 3-column cards, popular card has ring + shadow + badge
- **Footer:** 4-column grid, copyright

### 2. Login (/login)
- Centered single-column max-w-sm
- GitHub + Google social buttons (official SVG logos)
- Divider "Or continue with email"
- Magic link email form
- Post-send confirmation state

### 3. Signup (/signup)
- Same layout as login
- Benefit bullets (free credits, no CC, 60s deploy)
- Same social + magic link flow

### 4. Dashboard (/dashboard)
- Sidebar layout: collapsible sidebar (desktop) + bottom tab bar (mobile)
- Overview: 3 stat cards + credit progress bar + area chart + instance list
- Stat cards: label, value, sublabel, icon, optional trend

### 5. Instances (/dashboard/instances)
- Instance list: card per instance with status badge, URL, credit usage
- Create new: inline card form with name input
- Empty state with CTA

### 6. Billing (/dashboard/billing)
- Current plan card with manage button
- Credit balance with progress bar
- Top-up packages: 3-column grid (Small/Medium/Large)
- Plan comparison: 3-column, current plan highlighted
- Invoice history: list with download button

### 7. Settings (/dashboard/settings)
- Profile form (name, email)
- API key: masked display with show/hide/copy
- Danger zone: red border card, delete confirmation with text input

## Amplitude Events

| Event | Page/Component | Properties |
|-------|---------------|------------|
| page_view | All pages | { service, page } |
| signup_started | Hero CTA, Signup page | { service, source } |
| signup_completed | Signup page | { service, method } |
| login | Login page | { service, method } |
| feature_used | Dashboard, Instances, Settings | { service, feature } |
| pricing_viewed | Hero CTA, Billing page | { service, source } |
| checkout_started | Pricing, Billing page | { service, plan/action/package } |

All events include `service: "paperclipweb"` parameter for cross-project filtering.
