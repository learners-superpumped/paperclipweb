## Critical 수정 검증

### 1. POST /api/proxy/ai — VERIFIED

파일: `src/app/api/proxy/ai/route.ts`

- Anthropic/OpenAI 라우팅: `getProviderFromModel()` 함수가 모델명 prefix(`claude`, `anthropic`)로 분기. 나머지는 OpenAI.
- 크레딧 잔액 선확인: `credits.balance < 1` 이면 즉시 402 반환.
- AI API 호출: `callAnthropicAPI()` / `callOpenAIAPI()` 각각 실제 외부 API 호출.
- 크레딧 차감: `addCreditTransaction({ type: "usage", amount: -1, provider, model, tokensInput, tokensOutput })`.
- 레이스컨디션 처리: `txResult.success === false` 이면 다시 402 반환.
- Amplitude 트래킹: `trackServerCreditsUsed()` 호출.
- 환경변수: `ANTHROPIC_API_KEY`, `OPENAI_API_KEY` 모두 참조.

### 2. GitHub + Google OAuth — VERIFIED

- `import Google from "next-auth/providers/google"` 추가됨.
- `providers` 배열에 `GitHub({ clientId: process.env.AUTH_GITHUB_ID, clientSecret: process.env.AUTH_GITHUB_SECRET })` 포함.
- `providers` 배열에 `Google({ clientId: process.env.AUTH_GOOGLE_ID, clientSecret: process.env.AUTH_GOOGLE_SECRET })` 포함.
- 기존 이메일 매직링크 프로바이더 유지.

### 4. 크레딧 잔액 음수 방지 — VERIFIED

파일: `src/lib/queries.ts`, `addCreditTransaction()` (99번~157번 줄)

- `data.amount < 0` 인 경우 먼저 `getUserCredits()` 재조회.
- `userCredits.balance < Math.abs(data.amount)` 이면 `{ success: false }` 반환하고 DB에 쓰지 않음.
- 잔액이 충분할 때만 트랜잭션 INSERT → 잔액 UPDATE 실행.
- 추가: 크레딧 잔액이 한도의 10% 이하가 되면 `sendCreditLowEmail()` 발송. 이전 경고(Warning 6 — 데드코드)도 동시에 해소됨.
