## 3. 무료/유료 정책 설계

### 원가 기반 크레딧 역산

1 에이전트 액션의 원가가 ~$0.014이므로, 각 플랜의 포함 크레딧을 "구독료 대비 적절한 마진이 나는 수준"으로 설계해야 한다.

**목표 마진율: 60%+ (API 비용 마크업)**

마크업 계산: 원가 $0.014 x 2.5배 마크업 = 유저 관점 액션 가치 $0.035

이렇게 하면:
- $19 구독료에서 API 크레딧으로 배분 가능한 금액: ~$12 (나머지 $7은 인프라+마진)
- $12 / $0.014 원가 = **~857 액션 제공 가능**
- 반올림하여 **1,000 액션** 제공 (원가 $14, 구독료에서 배분 $12 + 약간의 마진 압박 감수)

## 4. 크레딧 정책

### 1 크레딧의 정의

**1 크레딧 = 1 에이전트 액션 = AI 모델에 1회 호출 (평균 2,500 입력 토큰 + 1,000 출력 토큰)**

구체적으로:
- 에이전트가 이메일 1통을 작성하면: 1 크레딧
- 에이전트가 데이터 분석 1건을 수행하면: 1 크레딧
- 에이전트가 코드 리뷰 1건을 수행하면: 1-3 크레딧 (복잡도에 따라)
- 에이전트 간 대화 1턴: 1 크레딧 (각 에이전트별)

### 기타 정책

- **크레딧 이월:** 불가 (매월 1일 리셋)
- **크레딧 양도:** 불가
- **미사용 환불:** 불가
- **모델 선택:** 유저가 에이전트별로 모델 선택 가능 (Sonnet/Haiku/GPT-4o/GPT-4o-mini). 크레딧은 모델 무관하게 1 액션 = 1 크레딧으로 통일 (내부적으로 모델별 원가 차이는 운영 측에서 흡수)

> **모델 무관 1크레딧 정책의 이유:** 유저 경험을 단순화한다. "Sonnet을 쓰면 3크레딧, Haiku를 쓰면 1크레딧" 같은 복잡한 체계는 BYOK의 복잡성을 재현하게 되어, "원빌링의 단순함"이라는 핵심 가치를 훼손한다. 대신, 경량 모델 우선 라우팅으로 원가를 관리한다.

> **문제:** 10:1 비율에서는 무료 유저 원가를 커버하지 못한다.

### 무료 유저 원가 최적화

무료 유저의 실제 행동을 고려한 조정:
- 가입 후 실제로 활성 사용하는 비율: ~30% (나머지 70%는 가입만 하고 떠남)
- 비활성 유저의 월 원가: $0 (컨테이너 미생성 또는 자동 중지)
- 활성 무료 유저의 평균 사용량: 100 액션 중 50 액션 (절반만 사용)

**조정된 무료 유저 평균 원가: $0.72/유저** (30% 활성 x $2.40)

| 항목 | Starter | Pro |
|------|---------|-----|
| 유료 월 마진 | $8.00 | $21.00 |
| 무료 평균 원가 (조정) | $0.72 | $0.72 |
| **커버 비율 (조정)** | **유료 1명 = 무료 11.1명** | **유료 1명 = 무료 29.2명** |

### 손익분기점 (BEP)

| 항목 | 금액 |
|------|------|
| 월 고정비 | $101.25 |
| Starter 순마진 (무료 배분 후) | $8.00 - ($0.72 x 9) = $1.52 |
| Pro 순마진 (무료 배분 후) | $21.00 - ($0.72 x 9) = $14.52 |

**유료 유저 믹스 기준 BEP** (Starter 80% : Pro 20% 가정):

- 가중 평균 순마진: ($1.52 x 0.8) + ($14.52 x 0.2) = $1.22 + $2.90 = **$4.12/유료유저**
- **BEP: 101.25 / 4.12 = 약 25명의 유료 유저**
- 10:1 비율이면 총 유저 275명 (유료 25 + 무료 250)

### BEP 도달 타임라인

스펙의 가설 기준:
- 1달: 유료 20명 -> BEP 근접 ($82.4 마진 vs $101.25 고정비) -- **미도달**
- 3달: 유료 80명 -> **충분히 BEP 초과** ($329.6 마진 vs $101.25 고정비)
- **현실성: 현실적** -- 3개월 내 BEP 도달 가능

### 왜 이 가격인가

1. **$19 Starter**: Railway $5 + API $20-40 = 실질 $25-45 대비 저렴. PaperclipCloud Hobby $15보다 $4 비싸지만 번들 크레딧 1,000회 포함 (PaperclipCloud는 BYOK). 원빌링 프리미엄.

2. **$49 Pro**: PaperclipCloud Entrepreneur $69 대비 $20 저렴하면서 번들 크레딧 3,000회 포함. DIY 총비용($25-45)과 PaperclipCloud($69) 사이의 sweet spot.

3. **무료 100 액션**: 에이전트 2-3개를 며칠 돌려볼 수 있는 양. "써보고 결정"을 실현하되, 원가를 $3 이내로 제한.
