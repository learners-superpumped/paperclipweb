# Development Report: paperclipweb MVP
**Date:** 2026-03-31
**Developer:** @product-developer

## Summary
Full-stack MVP implementation for paperclipweb - a bundled-credit Paperclip hosting service. Built on top of the designer's existing UI components and page layouts.

## Code Review Fixes (2026-03-31)

### Critical Fix 1: POST /api/proxy/ai (MVP core)
- Created `src/app/api/proxy/ai/route.ts`
- Accepts `{ model, messages, instance_id }`, validates with Zod
- Checks auth and credit balance (returns 402 if insufficient)
- Routes to Anthropic or OpenAI based on model prefix
- Deducts 1 credit via `addCreditTransaction({ type: "usage" })`
- Tracks `credits_used` event via server-side Amplitude
- Returns AI response with token usage metadata

### Critical Fix 2: GitHub + Google OAuth
- Added `GitHub` and `Google` providers to `src/lib/auth.ts`
- Uses `AUTH_GITHUB_ID`, `AUTH_GITHUB_SECRET`, `AUTH_GOOGLE_ID`, `AUTH_GOOGLE_SECRET` env vars
- Added env vars to `.env.example`

### Critical Fix 3: Dynamic company routes
- Created `src/app/api/companies/[id]/route.ts` (GET single, DELETE)
- Created `src/app/api/companies/[id]/usage/route.ts` (GET credit usage)
- Removed query-param DELETE from main companies route
- All routes verify ownership (userId match)

### Critical Fix 4: Credit balance negative prevention
- `addCreditTransaction()` now checks `balance >= |amount|` before deduction
- Returns `{ success: false, error: "Insufficient credits" }` if insufficient
- `/api/proxy/ai` checks balance before AI call AND handles race condition after

### Warning Fix: Delete Account
- Added `DELETE` handler to `/api/user/route.ts`
- Added `deleteUser()` query function
- Wired Delete Account button in settings page to call `DELETE /api/user`

### Warning Fix: sendCreditLowEmail() wired
- Now triggers inside `addCreditTransaction()` when balance drops below 10% of limit
- No longer dead code

### Schema Enhancement
- Added `provider`, `model`, `tokens_input`, `tokens_output` columns to `credit_transactions`

### New Env Vars Required
- `ANTHROPIC_API_KEY` - for Claude model proxying
- `OPENAI_API_KEY` - for GPT model proxying
- `AUTH_GITHUB_ID` / `AUTH_GITHUB_SECRET` - GitHub OAuth
