## 1. 시장 규모 (보수적)

### TAM (Total Addressable Market)
- 산출 방식: bottom-up
- Paperclip GitHub Stars: ~42K (4주 만에), 884개 활성 repos
- AI Agent 오케스트레이션 시장: $10.9B (2026, Grand View Research)
- 호스팅/관리형 서비스 하위 시장 (전체의 ~5%): ~$545M
- 타겟 유저: Paperclip을 사용하는 1인/소규모 창업자 (셀프호스팅 피로감)
- 추정 Paperclip 활성 사용자: 42K stars의 10% 실사용 = ~4,200명
- 잠재 확장 (2026년 말): AI agent 시장 성장률 적용 시 ~10,000명
- 유저당 연간 지불 가능액: $228 (Starter $19/mo x 12)
- **TAM: $2,280,000 (연간, Paperclip 호스팅 하위 시장)**

### SAM (Serviceable Available Market)
- TAM에서 제한 적용:
  - 영어권만 (70% 제외 -> 30%)
  - 웹 기반 호스팅 선호 (셀프호스팅 선호자 제외, 50%)
- **SAM: $342,000 (TAM의 15%)**

### SOM (Serviceable Obtainable Market) -- 보수적
- 첫해 현실적 점유율: SAM의 5%
- Paperclip 공식 호스팅(PaperclipCloud)이 이미 존재하므로 진입 장벽 있음
- **SOM: $17,100 (연간)**
- 월간 환산: ~$1,425/월
- 유료 유저 환산: ~75명 (연평균, Starter 기준)
- 근거: PaperclipCloud 출시 후 커뮤니티 반응 & Railway/Zeabur 셀프호스팅 가이드의 인기 기반

**보수적 원칙 적용:**
- 위 수치는 낙관적 추정의 1/3로 이미 조정됨
- 바이럴/입소문 계수: 0으로 가정
- PaperclipCloud의 번들 크레딧 "coming soon" 출시 시 50% 감소 리스크 반영하지 않음 (최악 시나리오는 Kill 기준에서 처리)

   - 현재 가정: 125명/주 (8% 성장)
   - 목표: 500명/주 (1주차)
   - 레버: Paperclip GitHub README에 링크 추가, Discord 커뮤니티 홍보, Reddit/HN 포스트
   - 영향: 방문자가 2배면 전체 퍼널이 2배

   - 연관 가설: VH1

## 7. Kill/Continue 기준 (Pre-PMF)

### 1주 체크: 성장 가설 초기 검증

**GH1 검증: 방문자 수**
- VALIDATED: 방문 500명 이상 -> 커뮤니티 채널 유효, Continue
- INCONCLUSIVE: 방문 100-499명 -> 채널은 맞지만 메시지 개선 필요
- INVALIDATED: 방문 100명 미만 -> 채널 피봇 (Reddit/HN/ProductHunt 등 대안)

**VH1 초기 신호: 가입 수**
- VALIDATED: 가입 30명 이상 -> 번들 크레딧 메시지 유효
- INCONCLUSIVE: 가입 10-29명 -> 랜딩페이지 개선
- INVALIDATED: 가입 10명 미만 -> 가치제안 재검토

**Kill 기준: 방문 50명 미만 AND 가입 5명 미만** -> 시장 자체가 반응하지 않음

### 1달 체크: 가치 가설 본검증

**VH1 검증: 누적 가입 수**
- VALIDATED: 가입 200명 이상 -> 번들 크레딧 수요 확인
- INCONCLUSIVE: 가입 50-199명 -> 메시지/채널 개선 후 재검증
- INVALIDATED: 가입 50명 미만 -> 원빌링 가치가 없거나 타겟이 다름

**VH2 검증: 유료 전환**
- VALIDATED: 유료 20명 이상 (전환율 10%+) -> PMF 강력 신호
- INCONCLUSIVE: 유료 5-19명 -> 가격/기능 조정 후 재검증
- INVALIDATED: 유료 5명 미만 -> 가격 피봇 또는 Kill

**Kill 기준: 유료 3명 미만 AND MRR < $57** -> 제품에 돈을 낼 가치가 없음

### 3달 체크: PMF 판단

**종합 판단 매트릭스:**

| VH1 (가입) | VH2 (유료) | GH1 (방문) | 판단 |
|-----------|-----------|-----------|------|
| VALIDATED | VALIDATED | VALIDATED | PMF 달성 -> Scale 단계 |
| VALIDATED | VALIDATED | INVALIDATED | PMF 있으나 채널 변경 필요 |
| VALIDATED | INVALIDATED | - | 가격/기능 피봇 |
| INVALIDATED | - | - | Kill (가치 없음) |

**3달 Kill 기준:**
- 누적 가입 < 200명 (목표 1,000의 20%)
- 유료 < 15명 (목표 80의 19%)
- MRR < $285 (BEP $101.25의 2.8배 미만)
- 월간 성장률 < 5% (정체)

**3달 Continue 기준:**
- 누적 가입 >= 500명
- 유료 >= 40명
- MRR >= $760
- 월간 유료 성장률 >= 15%
